## Overview

This project builds a **Car Sales Recommendation Engine** that generates intelligent sales insights and computes fair, data-driven **employee payout structures**. By combining probabilistic modeling, forecasting, and economic theory, the engine moves beyond gut-feel sales management toward a rigorous, quantitative framework.

## Dataset

**Source:** [USA Car Sales Dataset 2018–2024 — Kaggle](https://www.kaggle.com/datasets/anjaliprajapati307/usa-car-sales-dataset-2018-2024/versions/1)

This dataset provides a rich, multi-dimensional view of individual car sales transactions, capturing information across:

| Dimension | Examples |
|---|---|
| **Customer Demographics** | Age, income bracket, region |
| **Car Specifications** | Make, model, year, category |
| **Pricing Metrics** | MSRP, sale price, dealer margin |
| **Payment Details** | Financing type, loan term, down payment |
| **Sales Performance** | Rep ID, time-to-close, units sold |
| **Seasonal & Regional Context** | Month, quarter, state, dealership |

Each row represents a **single car sale transaction**, enabling granular analysis of how various factors drive profitability, sales patterns, and customer behavior over a **6-year horizon (2018–2024)**.
